# <p align="center">  Requisitos do Sistema – Circuito Verde Magé</p>

### ✅ Requisitos Funcionais (o que o sistema deve fazer)

* **RF01:** Exibir lista de pontos turísticos, trilhas, cachoeiras e eventos.
* **RF02:** Permitir busca e filtros (por tipo de atração, categoria, localização).
* **RF03:** Mostrar informações detalhadas de cada ponto.
* **RF04:** Área administrativa com login para gestores.
* **RF05:** Administradores podem cadastrar, editar e remover pontos turísticos e eventos.
* **RF06:** Cadastro/Login (incluindo login via Google).
* **RF07:** Exibir mapa com pontos turísticos oficiais.
* **RF08:** Álbum *“Minhas Conquistas”* para exibir os locais visitados.
* **RF09:** Permitir compartilhamento de atrações cadastradas em redes sociais.

---

### ⚙️ Requisitos Não Funcionais (como o sistema deve ser)

* **RNF01:** O app deve ter interface intuitiva e responsiva.
* **RNF02:** O app deve responder rapidamente mesmo com muitos usuários simultâneos.
* **RNF03:** Dados de login e administradores devem ser armazenados com segurança (criptografia).
* **RNF04:** O app deve funcionar parcialmente offline (modo básico, mapas e informações salvas).
* **RNF05:** O app deve possuir baixo consumo de internet e bateria.
* **RNF06:** O app deve possuir escalabilidade para futuras integrações.
* **RNF07:** O app deve ser compatível com Android e iOS.
* **RNF08:** O app deve possuir versão web para desktops.
